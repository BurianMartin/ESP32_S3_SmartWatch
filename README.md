# Bachelor's Thesis – Smartwatch Software Interface using ESP32

This repository contains the source code for the bachelor's thesis:
**"Design and Implementation of a Software Interface for a Smartwatch using ESP32"**

## Project Overview

A complete smartwatch OS implementation for the T-Watch S3 (ESP32-S3), 
built as part of a bachelor's thesis at VŠB – Technical University of Ostrava.
Includes full source code, configuration files, and build instructions.

## Requirements

- **GCC/G++** – C/C++ compiler
- **Visual Studio Code**
- **PlatformIO extension** for VSCode

## Build Instructions

1. Open Visual Studio Code
2. Open the project folder containing `platformio.ini`
3. PlatformIO will automatically initialize the project on first open
4. Click the PlatformIO icon in the left panel and select **"Build"**  
   or press `Ctrl + Alt + B` (Windows/Linux) / `Cmd + Alt + B` (macOS)

## Flashing to Device

After a successful build, upload to your ESP32 / T-Watch S3 via the 
**"Upload"** button in PlatformIO. Connect the device via USB first.
